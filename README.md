# 🚀 Advanced Personal Portfolio Website — Bivan Jena

A modern, responsive, and professional portfolio website built with HTML5, CSS3, and JavaScript (ES6). This project showcases frontend development skills through a clean, animated, and interactive user interface.

![Portfolio Preview](assets/images/profile.png)

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎨 **Dark Mode** | Toggle between light/dark themes with localStorage persistence |
| ⌨️ **Typing Animation** | Dynamic text effect using Typed.js in the hero section |
| 📜 **Scroll Animations** | Smooth reveal animations powered by AOS library |
| ⬆️ **Back to Top** | Floating button that appears after scrolling |
| 📱 **Fully Responsive** | Mobile, tablet, and desktop layouts |
| 📊 **Animated Skill Bars** | Progress bars animate on scroll into view |
| 📬 **Contact Form** | Client-side validation with success toast notification |
| 🧭 **Smooth Navigation** | Active link highlighting and smooth scrolling |
| 🍔 **Hamburger Menu** | Animated mobile menu with overlay |

---

## 🛠️ Technologies Used

### Core
- **HTML5** — Semantic structure
- **CSS3** — Custom properties, Flexbox, Grid, animations
- **JavaScript (ES6)** — DOM manipulation, Intersection Observer, localStorage

### Libraries (CDN)
- **[AOS](https://michalsnik.github.io/aos/)** — Scroll animations
- **[Typed.js](https://mattboldt.com/demos/typed-js/)** — Typing effect
- **[Font Awesome 6](https://fontawesome.com/)** — Icons
- **[Google Fonts](https://fonts.google.com/)** — Inter & Outfit typefaces

## 📋 Sections

1. **Navigation Bar** — Fixed glassmorphism navbar with dark mode toggle
2. **Hero** — Name, typed roles, CTA buttons, social links
3. **About Me** — Bio, education, stat cards
4. **Skills** — 4 categories with animated progress bars
5. **Projects** — 3 project cards with hover overlay
6. **Achievements** — Certifications and milestones
7. **Services** — Web Dev, UI Design, Backend Dev
8. **Contact** — Form with validation + contact info
9. **Footer** — Copyright and social links

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/bivanjena/portfolio-website.git
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   # Or use Live Server in VS Code
   ```

3. **Customize**
   - Update personal info in `index.html`
   - Add your `resume.pdf` to `assets/`
   - Update social media links
   - Replace project images with your own

## 📱 Responsive Breakpoints

| Breakpoint | Layout |
|---|---|
| `> 1024px` | Full desktop layout |
| `768px – 1024px` | Two-column tablet layout |
